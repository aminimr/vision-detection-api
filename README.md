# Vision Detection API

API تشخیص و بخش‌بندی اشیاء با YOLOv8 و FastAPI

## ویژگی‌ها

- تشخیص و بخش‌بندی اشیاء با YOLOv8
- پشتیبانی از 5 مدل مختلف (Nano تا XLarge)
- بصری‌سازی خودکار با ماسک و باکس
- API کامل با FastAPI
- پشتیبانی از آپلود و URL
- پردازش دسته‌ای
- Docker و Docker Compose
- کش با Redis
